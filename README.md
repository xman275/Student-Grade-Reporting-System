**Description:** Developed a robust Java application designed to automate the calculation, management, and reporting of academic performance metrics for educational environments.

**Tech Stack:** Java, Object-Oriented Programming (OOP), File I/O, Data Structures.

**Architectural Design:** Utilized Object-Oriented Programming (OOP) principles, including encapsulation and inheritance, to create a scalable system for managing student and course data.

**Data Processing:** Implemented complex algorithms to calculate weighted averages, GPA, and letter grades based on customizable academic rubrics and criteria.

**File Integration:** Engineered File I/O functionality to securely read student records from external databases and export comprehensive, formatted grade reports for administrative use.
